pith
====

A simple command-line tool to execute Python while taking care of the
``PYTHONPATH``.

Project Homepage
================

.. _`pith Homepage`: https://github.com/weegreenblobbie/pith-tool

- `pith Homepage`_

How to Install
==============

.. code-block:: console

    pip install --user pith

Or:

.. code-block:: console

    pip3 install --user pith
